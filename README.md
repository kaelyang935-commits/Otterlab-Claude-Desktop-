## 📌 這是什麼？

這份教材教你 **Claude Desktop（桌面版 App）裡的斜線指令（slash commands）**。

一個你必須先知道的關鍵觀念：

> **這些 `/` 指令住在桌面 App 的「Code」分頁裡（Code 分頁 ＝ 帶圖形介面的 Claude Code）。**
> 桌面 App 有三個分頁：**Chat**（一般聊天）、**Cowork**（背景自動跑任務）、**Code**（寫程式／動手做專案）。本教材的 `/code-review`、`/diff`、`/model`… 都在 **Code 分頁**;在 Chat 分頁打 `/` 是看不到它們的。Code 分頁的指令,跟工程師在終端機執行 `claude` 看到的**幾乎完全相同**,差異只在少數「互動式面板」的呈現與網頁版限制——詳見 [第 5 課](docs/05-environments.md)。

> 🆕 **完全沒裝過、也不是工程師?** 先做 [第 0 課：安裝、登入、做出第一個成果](docs/00-getting-started.md),10 分鐘走到「能動手」再回來。

**先記住這四個名字,之後不會搞混：**

| 名稱 | 是什麼 |
|---|---|
| **Claude Desktop** | 你下載的桌面版 App（本教材的主場景） |
| **Code 分頁 ＝ Claude Code** | App 裡「寫程式／動手做專案」的分頁,斜線指令都在這 |
| **Cowork** | App 裡另一個分頁,把較長、較自動的任務丟給它在背景做 |
| **終端機 `claude`** | 工程師在黑底視窗用的同一套引擎(你不必另外裝) |

## 🚀 快速開始

> 完全沒裝過?先做 [第 0 課](docs/00-getting-started.md);已經能在 Code 分頁打字了,看這裡。

**🟢 非工程師、第一天只記這 5 件事：**

| # | 怎麼做 | 為什麼 |
|---|---|---|
| 1 | **直接用人話描述你要的東西** | 這才是主角：「幫我做一個…」「幫我整理這份…」 |
| 2 | `/help` | 看看有哪些指令可用 |
| 3 | **把檔案／圖片直接拖進對話** | 讓 AI 看你的資料再動手 |
| 4 | `/clear` | 想換主題時,開一段乾淨的新對話 |
| 5 | `/rewind` | **做錯了的後悔藥**：把對話／程式碼倒回先前某一點 |

**🔵 等你開始寫程式、或對話變很長,再學這 5 個：**

| 指令 | 一句話 |
|---|---|
| `/context` | 看目前用掉多少「上下文」（對話記憶） |
| `/compact` | 對話太長時,摘要壓縮以省空間 |
| `/model` | 切換要用哪個模型 |
| `/diff` | 檢視程式碼這次改了哪幾行 |
| `/usage` | 查看花費與用量 |

> 💡 在 **Code 分頁** 的輸入框打一個 `/`,就會跳出指令選單;繼續打字會即時篩選。

> 🛡️ **三條安全底線（任何人開始前必讀）：**
> 1. 要 AI 跑「別人的程式或技能」前,先請它「**只摘要、先別執行**」。
> 2. 不確定的網址或檔案,**不要叫它「照做」**。
> 3. 看到要**刪檔、部署、付費、寄信**這類動作,先停下來確認。
> （為什麼?見 [第 9 課](docs/09-run-skill-from-url.md)。）

## 🧭 學習路徑（兩條線,挑一條開始）

**🟢「我只想用 AI 做東西」（非工程師、在職快速上手）：**
```
第0課 安裝+第一個成果 ─▶ 第1課 指令觀念 ─▶ 第2課「快速開始 5 個」
   ─▶ 第9課 給一個 URL 用現成技能(零安裝,新手最有感)
   ─▶ 想做自己的小指令? 看第4課「30 秒純文字版」就好
```

**🔵「我要寫程式／帶團隊」（開發者）：**
```
第0課 ─▶ 第1課 ─▶ 第2課 內建指令全覽 ─▶ 第3課 技能指令
   ─▶ 第4課 自訂指令(frontmatter/注入) ─▶ 第5課 環境差異
   ─▶ 第7課 官方 Skills(別替換) ─▶ 第8課 找 .md ─▶ 第9課 一鍵執行
```

> 💡 標「進階／開發者向」的第 3、4、7、8 課,非工程師可先略過,日後想自己擴充工具時再回來。

## 📖 如何使用本教材

**給學生：**
1. 完全沒裝過,先做 [第 0 課](docs/00-getting-started.md);再從 [第 1 課](docs/01-fundamentals.md) 照順序讀。
2. 每課結尾都有「課堂練習」，務必親手打一次指令。
3. 把 [速查表](CHEATSHEET.md) 印出來放在手邊。

**給講師：**
1. 直接看 [第 6 課 講師教學手冊](docs/06-instructor-guide.md)，裡面有授課順序、時間分配、常見誤區與評量題。
2. 每課的「課堂練習」可當作隨堂演練或回家作業。
3. 速查表可作為課後發放的講義。
