# 第 0 課｜安裝、登入,做出你的第一個成果（0→1 起點）

> 回到：[課程首頁](../README.md) ｜ 下一課：[第 1 課 基礎概念](01-fundamentals.md)

> 本課給「完全沒用過、也不是工程師」的你：把環境裝好、登入、認識畫面,並在 10 分鐘內用「**一句話**」做出第一個看得到的成果。先有成就感,再學指令。

---

## 🎯 學習目標

讀完本課,你能夠：

1. 安裝並登入 Claude 桌面 App。
2. 認識畫面：**Chat ／ Cowork ／ Code** 三個分頁,知道斜線指令在哪。
3. 選一個資料夾當「工作區」。
4. **用一句自然語言,做出你的第一個網頁。**

---

## 1. 先搞懂：什麼是「Vibe Coding」?

你**不需要先學程式語言**。Vibe coding 就是：用「人話」描述你要的東西,讓 AI 幫你寫程式、做檔案、跑起來;你只負責「**想要什麼**」與「**驗收**」。

本課程教你怎麼跟這位 AI 夥伴(我們叫它 **agent**)合作。

> 🧠 **agent（AI 代理）：** 會「自己動手做事」的 Claude——你給它一個目標,它會自己讀檔、寫檔、跑指令、用各種工具一步步完成,而不只是回你一句話。本課程裡的 Claude Desktop 就是一個 agent。

---

## 2. 安裝與登入

1. **下載：** 到官方下載頁取得桌面版 App（macOS／Windows）：
   - 下載：<https://claude.com/download>
   - 圖解步驟：<https://code.claude.com/docs/en/desktop-quickstart>
2. **安裝後開啟**,用 Google 或 Email **登入** Anthropic 帳號（對應的指令是 `/login`）。
3. 確認右上角顯示你的帳號 = 已登入。

> 💳 **在職學員請先知道：** 本課程用到的「Code 分頁／斜線指令」功能,需要 **Pro／Max／Team／Enterprise** 其中一種付費方案。純免費的聊天可以體驗,但要「動手做專案」建議先確認方案。費用觀念見 [第 2 課的「💰 關於費用」](02-builtin-commands.md)。

---

## 3. 認識畫面：三個分頁（**最關鍵**）

打開桌面 App,上方有三個分頁,用途完全不同：

| 分頁 | 做什麼 | 本課程用到嗎 |
|---|---|---|
| **Chat** | 一般聊天問答（像平常用 ChatGPT） | ➖ |
| **Cowork** | 把較長、較自動的任務交給它在背景做 | ➖（進階） |
| **Code** | 寫程式／動手做專案的地方 | ✅ **本課程的斜線指令都在這** |

> ⚠️ **新手第一個常見卡關：** 打 `/` 沒反應、找不到 `/code-review`、`/diff`?——**先確認你在 Code 分頁。** 在 Chat 分頁打 `/` 是看不到這些指令的。

「Code 分頁」其實就是帶圖形介面的 **Claude Code**：有側邊欄、檔案編輯器、視覺化的程式碼差異(diff)、即時預覽你做出來的網頁/App。本課程後面講的指令,跟工程師在終端機打 `claude` 用的是**同一套**。

---

## 4. 選一個「工作區」資料夾

在 **Code 分頁**,新建或開啟一個資料夾當你的工作區（例如在桌面建一個 `my-first-app`）。

> 這個資料夾就是 AI 幫你「建立檔案」的地方。沒有它,AI 只能跟你聊天、不能幫你產出檔案。

---

## 5. 🚀 你的第一個成果：一句話做一個網頁

在 Code 分頁的輸入框,貼上這句話：

```
幫我做一個單頁自我介紹網頁：有我的名字、三個興趣、漸層背景。
做好後幫我在瀏覽器打開給我看。
```

按下送出,看 agent 動手：

> ✅ **你應該會看到：** agent 建立一個 `index.html` 檔 → 幫你打開預覽 → 一個有你名字的網頁出現了。

不滿意?**直接用人話改**,例如：

```
背景改成深色、名字字體大一點、再加一個聯絡用的 Email。
```

它會幫你改好、重新打開。**這就是 vibe coding——你出嘴,它出手。**

---

## 6. 🛡️ 開始前必讀：三條安全底線

從今天第一天就請記住（之後 [第 9 課](09-run-skill-from-url.md) 會詳細解釋為什麼）：

1. **要 AI 跑「別人的程式或技能」前,先請它「只摘要、先別執行」。**
2. **不確定的網址或檔案,不要叫它「照做」。**
3. **看到要刪檔、部署、付費、寄信這類動作,先停下來確認。**

> 🧠 比喻：把網路上一份陌生說明書交給你的助理、還說「上面寫什麼你都照做」是很危險的——那份說明書可能叫他刪你的檔、傳你的資料。所以**先看過、或先請它摘要**。

## 📝 課堂練習

> 請在 Claude Desktop 實際操作。

1. **找位置：** 進到 **Code 分頁**,打一個 `/`,看看跳出哪些指令。（在 Chat 分頁打 `/` 比較看看,有什麼不同?）
2. **做出成果：** 用第 5 節那句話做出你的第一個網頁,再用「一句話」改它**一個地方**。
3. **認環境：** 在畫面上找出「目前登入的帳號」和「目前用哪個模型」分別在哪裡。

## 🔑 重點回顧

- 下載登入後,**桌面 App 有三分頁：Chat／Cowork／Code;斜線指令在 `Code` 分頁。**
- Vibe coding = 用人話「描述 + 驗收」,agent 幫你動手做。
- **三條安全底線從第一天就記住。**
